# Privacy Policy

*Last updated: April 2026*

## 1. Data Collection

RADBot does not collect, store, or share any personal data. No user information is logged, retained, or processed beyond what is strictly necessary for the bot to function during an active session.

Discord User IDs are considered personal data under GDPR (Art. 4). RADBot does not store, log, or process Discord User IDs beyond the duration of a single interaction.

## 2. Voice & Text Channels

RADBot only reads slash commands directed at it. It does not read, store, or process any other messages, voice data, or user content.

## 3. In-Memory Session Data

RADBot temporarily holds the following data in memory solely to enable audio playback. This data is never written to disk and is permanently deleted when the bot restarts or leaves a server:

- Guild ID and voice channel ID (required to maintain the audio connection)
- Currently playing station per server
- Volume setting per server
- Pause/resume state per server

No user-identifiable information is stored at any point.

## 4. Legal Basis

The temporary processing of guild and channel IDs is based on legitimate interest (GDPR Art. 6 para. 1 lit. f) to provide the requested radio streaming functionality. No data is retained after the session ends.

## 5. Third Party Services

RADBot streams audio from publicly available radio stations and uses the [Radio Browser API](https://www.radio-browser.info) to search for stations. We are not responsible for the content or availability of third party streams. The Radio Browser API does not receive any personal user data from RADBot.

RADBot is listed on the following third party bot directories:

- [top.gg](https://top.gg)
- [discord.bots.gg](https://discord.bots.gg)
- [discordbotlist.com](https://discordbotlist.com)

These services may collect data independently. Please refer to their respective privacy policies.

## 6. Logging

RADBot logs technical events (errors, stream health checks) to local log files on the host server for debugging purposes only. These logs do not contain any user-identifiable information such as User IDs, usernames, or message content.

## 7. Data Retention

RADBot retains no personal data. All session data is held exclusively in memory and is permanently deleted upon bot restart. No databases, files, or external services are used to store user data.

## 8. Your Rights (GDPR)

As RADBot stores no personal data, the rights to access, rectification, erasure, and portability (GDPR Art. 15–20) are satisfied by default. If you have any concerns, please contact us directly.

## 9. Children's Privacy

RADBot does not knowingly collect data from users under the age of 13. As RADBot collects no personal data at all, it is safe for use in any Discord server that complies with Discord's own Terms of Service.

## 10. Changes to This Policy

This privacy policy may be updated occasionally. The latest version is always available at this URL. Continued use of RADBot after changes constitutes acceptance of the updated policy.
