RADBot

Your worldwide radio bot — stream 30,000+ stations from every corner of the globe directly in your Discord server.

 Features
- 30,000+ stations from over 150 countries via Radio Browser API
- Browse by **genre (Jazz, Rock, Techno, Hip Hop, Classical and more)
- Browse by region (Europe, Americas, Asia, Africa, Middle East)
- Smart search — filter by station name, country and genre simultaneously
- Live track info via ICY metadata, updated every 15 seconds
- Shuffle — instant random station from the entire library
- Full **playback controls** — pause, resume, stop, volume (0–100%)

Commands
| Command | Description |
|---|---|
| `/radio` | Browse and play stations by category |
| `/search` | Search 30,000+ stations by name, country or genre |
| `/shuffle` | Play a random station |
| `/volume` | Set volume from 0 to 100% |
| `/title` | Show the currently playing track |
| `/stop` | Stop radio & leave voice channel |
| `/help` | Show all commands |

Search Examples

/search query:BBC
/search tag:jazz country:US
/search tag:rock country:germany
/search query:polska
